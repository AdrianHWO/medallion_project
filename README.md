# Medallion Architecture Data Pipeline

This repository contains an end-to-end data pipeline built on **Databricks** using **PySpark** and **Delta Lake**.

## Key Features
- **Bronze Layer**: Raw data ingestion into Delta tables.
- **Silver Layer**: Data transformation, cleaning, and filtering high-value transactions.
- **Gold Layer**: Business-level aggregations to provide actionable insights.

## Project Context
This project was developed as part of my transition into **Data Engineering**, focusing on scalable data infrastructure and cloud-native analytics.
